Four Steps to EXE

📁

Place Script

Drop py_builder.pyw into your project folder alongside your Python files

🔍

Auto-Detect

Scripts appear instantly in the list. The folder watcher refreshes in real time.

⚡

One Click

Select your script and press BUILD EXE. Dependencies auto-install if needed.

✅

Done

Your executable appears in the same folder. Temporary files auto-clean if enabled.

Get Started

Requirements

Windows 10/11 · Python 3.10+

# 1. Clone the repo
git clone https://github.com/jlaiii/Py-Builder-Automated-Python-to-EXE-GUI.git
cd Py-Builder-Automated-Python-to-EXE-GUI

# 2. Run — PyInstaller auto-installs on first launch
python py_builder.pyw

💡 Pro tip: Place py_builder.pyw in any project folder and double-click it. It auto-detects all scripts in that directory.
